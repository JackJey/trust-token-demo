Trust Token v1 DEMO

This site demonstrates Trust Token v1 issuance and redemption, using BoringSSL to create an issuance service.

How to run a DEMO

Access an URL Trust Token Redeemer DEMO with Chrome 87(Stable) and click a "click continue to read".
